AutoHotKey

posted on 12/05/15

Every one needs shortcuts and few handy softwares as per his need.And i am also not an exception. I came to know about AHK in my second semester,when i was a calculus project.

It is really very powerful tool for Microsoft Windows.It is a free, open-source custom scripting language, aimed at providing easy keyboard shortcuts(or hotkeys),software automation that allows users of most levels of computer skill to mange repetitive tasks as per their need in any Microsoft Windows application.

I am a big fan of this scripting language.It helped us to create a shape detecting GUI which detected the type of 2D shape(e.g- Triangle,Rectangle,Circle) you draw.

you can also learn this great language:autohotkey
